# One-stop Event Forum + Community Hub

This project is a frontend application designed for students to check events, upload details, interact with each other, and receive reminders. It serves as a comprehensive platform for managing and participating in community events.

## Features

- **Event Listing**: View a list of upcoming events.
- **Event Details**: Access detailed information about each event.
- **Event Upload**: Users can upload new events and share them with the community.
- **Community Hub**: Engage in discussions and forums related to events and community interests.
- **Reminders**: Set and manage reminders for events to stay updated.
- **User Profiles**: Manage user profiles and personal information.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (version 6 or higher)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd event-forum-frontend
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm start
```
The application will be available at `http://localhost:3000`.

### Building for Production

To create a production build, run:
```
npm run build
```
This will generate a `build` folder containing the optimized application.
